# measure_weight

Append a new weight observation to `inst/extdata/martysweight.xlsx`, re-run the spline model in `inst/simple_summ_plot/iter1.r`, and present the resulting plot.

## Arguments

The user provides the arguments after `/measure_weight`. Parse them as follows:

- **kg** (required): body mass in kg (e.g. `81.3`)
- **time_hours** (optional): hour of measurement (0-23)
- **time_minutes** (optional): minute of measurement (0-59)
- **fasted_hours** (optional, default: 8): hours since last meal
- **fasted_minutes** (optional, default: 0): additional minutes since last meal

**Time default logic:** When time_hours and time_minutes are NOT provided, capture a timestamp at the moment the skill is invoked (via `date` or Bash `$(date +%H:%M)`), then subtract 2 minutes from it. This accounts for the delay between stepping off the scale and typing the command.

Examples:
- `/measure_weight 81.3` — logs 81.3 kg, (now minus 2 min), 8h fasted
- `/measure_weight 81.3 7 0 10 0` — logs 81.3 kg at 07:00, 10h fasted
- `/measure_weight 81.3 6 30` — logs 81.3 kg at 06:30, 8h fasted

## Steps

### 1. Parse arguments

Extract `kg` from the first argument. Remaining positional args map to `time_hours`, `time_minutes`, `fasted_hours`, `fasted_minutes`. Validate that `kg` is a number between 40 and 200.

If `time_hours` and `time_minutes` are not provided, immediately run:
```bash
date +"%H %M"
```
to capture the current time, then subtract 2 minutes from it (handling the hour rollback if minutes < 2, e.g. 07:01 becomes 06:59).

### 2. Append row to Excel

**IMPORTANT:** R segfaults with multi-line `-e` scripts from bash on this system. Always use the Write tool to create a temp `.r` file, then source it.

```r

dat  <- read.xlsx(path, detectDates = TRUE)
new_row <- data.frame(
  Date                = Sys.Date(),
  Time_hours          = TIME_HOURS,
  Time_minutes        = TIME_MINUTES,
  Time_fasted_hours   = FASTED_HOURS,
  Time_fasted_minutes = FASTED_MINUTES,
  Mass                = KG_VALUE,
  Episode             = NA
)
dat <- rbind(dat, new_row)
write.xlsx(dat, path)
cat("Appended:", KG_VALUE, "kg on", as.character(Sys.Date()), "\n")
```
